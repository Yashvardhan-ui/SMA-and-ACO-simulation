Slime Mould Algorithm

The cell that designs railways

In 2010, Toshiyuki Nakagaki placed food at positions matching Tokyo train stations and introduced Physarum polycephalum at the centre. Within 26 hours the organism had recreated the Yamanote Line topology — optimising for efficiency and fault tolerance without a nervous system. Published in Science, this experiment demonstrated that unicellular organisms can solve combinatorial network design problems purely through chemical feedback.

The Slime Mould Algorithm (SMA) formalised by Li et al. (2020) captures this: agents sense trail gradients in three forward directions and steer toward stronger signals, depositing chemical that attracts other agents. The oscillation weight W simulates bio-oscillation between positive and negative feedback phases, automatically balancing exploration and exploitation.

// Position update equations
X(t+1) = Xbest + vb · (W · XA − XB) probability p
X(t+1) = vc · X(t) otherwise
// Oscillation weight — positive feedback (rank ≤ N/2)
W(i) = 1 + r · log((bF − S(i)) / (bF − wF) + 1)
// Oscillation weight — negative feedback (rank > N/2)
W(i) = 1 − r · log((bF − S(i)) / (bF − wF) + 1)

Ant Colony Optimization

Emergent intelligence from chemistry

Marco Dorigo introduced ACO in his 1992 doctoral dissertation as the first algorithm to solve the Travelling Salesman Problem through emergent collective intelligence. Individual ants have no global knowledge — each deposits pheromone proportional to path quality, biasing successors toward already-reinforced trails. The resulting positive feedback loop converges the colony on globally optimal solutions.

// Probabilistic city selection
P(i→j) = [τ(i,j)α · η(i,j)β] / Σ_k [τ(i,k)α · η(i,k)β]
// Pheromone update
τ(i,j) ← (1−ρ) · τ(i,j) + Σ Δτ(i,j)
Δτ(i,j) = Q / Lk if ant k used edge (i,j)

About the Project

PathMind is a full-stack bio-inspired pathfinding simulator built to demonstrate how two distinct swarm intelligence algorithms — the Slime Mould Algorithm and Ant Colony Optimization — solve the problem of finding shortest routes between multiple cities while navigating real-world obstacles like buildings and water bodies.

The project implements both algorithms from first principles: SMA runs as a genuine background computation with pixel-resolution agent simulation and gradient-based path extraction; ACO provides a live animated colony with real-time pheromone visualisation and convergence detection.
